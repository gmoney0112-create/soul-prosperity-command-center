1. UTM convention

Every paid and organic destination URL carries a UTM block. The block is the same shape everywhere; only the values change.

?utm_source={platform}
&utm_medium={medium}
&utm_campaign={campaign-name}
&utm_content={ad-id-or-post-id}
&utm_term={angle-or-keyword}
FieldAllowed valuesExample
utm_sourcemeta · tiktok · youtube · x · email · sms · ig-bio · tiktok-bio · fb-bio · yt-desc · directmeta
utm_mediumpaid · organic · email · sms · bio · partnerpaid
utm_campaignSP-COLD-EBOOK7 · SP-COLD-FREEBIE · SP-RETARGET-CART · SP-RETARGET-BUYER · SP-EMAIL-WF02SP-COLD-EBOOK7
utm_contentStable ad ID or post slugreel-mcchicken-v2
utm_termAngle name (matches ad-copy swipe)authority · story · pain · curiosity · direct
Lowercase + dashes. Never use spaces, underscores, or capitals in UTM values. SP-COLD-EBOOK7 is the one exception (campaign codes are uppercase for readability in reports).

3. Pixel/event parameters

Pass these alongside every conversion event so retargeting and value-based audiences work:

{
  content_ids: ["ebook7"],          // or "audio17", "paperback27", "course67", "lifetime497"
  content_name: "Soul Prosperity 7 eBook",
  content_type: "product",
  currency: "USD",
  value: 7,
  num_items: 1,
  // optional but recommended for matching
  email: "{{contact.email}}",       // hashed by the platform
  external_id: "{{contact.id}}"     // GHL contact id
}

Send email and external_id only on server-side conversions API (where you can hash properly), not in the browser pixel.

4. Thank-you-page firing

Each thank-you page fires the matching purchase_* event exactly once. Use a single shared snippet template:

// On thank-you page load:
window.fbq && fbq('track', 'Purchase', {
  content_ids: ['ebook7'],
  content_name: 'Soul Prosperity 7 eBook',
  content_type: 'product',
  currency: 'USD',
  value: 7,
});
window.fbq && fbq('trackCustom', 'purchase_ebook7', {
  currency: 'USD',
  value: 7,
});
window.ttq && ttq.track('CompletePayment', { value: 7, currency: 'USD' });
window.gtag && gtag('event', 'purchase', {
  transaction_id: '{{order_id}}',
  value: 7, currency: 'USD',
  items: [{ id: 'ebook7', name: 'Soul Prosperity 7 eBook', price: 7, quantity: 1 }],
});

Replicate the block on every thank-you page, swapping ebook7/7 for the matching product/value.
